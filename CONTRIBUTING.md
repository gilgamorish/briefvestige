# Contributing to Vestige

Word bank contributions are the most valuable thing you can add.

## How to contribute

1. Fork the repo
2. Open `index.html`
3. Find the relevant word bank array in the `const W = {` object
4. Add your entries following the format below
5. Open a pull request with a short description of what you added and why

## Entry format guidelines

**Jobs** — plain role name, no descriptor needed
```
"siege engineer", "bone setter", "well digger"
```

**Races** — name followed by a dash and a short visual description
```
"ironclad — living metal grafted to bone, cold to touch, no expressions"
```

**Themes** — if it's an object or animal, add a dash and key visual qualities
```
"crow — black iridescence, scavenger logic, bone and wire"
```

**Cultures** — name followed by a dash and 4-6 specific visual references
```
"viking norse — furs, iron, carved wood, longship shapes, knotwork"
```

**Props** — be specific. avoid vague entries like "a sword" — prefer entries that tell a story
```
"snapped blade, kept anyway"
"flare gun with no flares — carried anyway"
```

## What makes a good entry

- specific enough that an artist knows what to draw
- not redundant with something already in the list
- weird is good — the unexpected combinations are the point
- if it's vague without context, add a descriptor

## What to avoid

- entries that are just one word with no visual direction
- duplicates of existing entries
- anything that requires explanation to understand
