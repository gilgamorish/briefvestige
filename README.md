# ◆ Vestige — Brief Generator

A concept art brief generator for character designers and environment artists. Built to get you to a pencil faster.

**Live site:** [conceptvestige.netlify.app](https://conceptvestige.netlify.app)

---

## What it does

Vestige generates structured design briefs — the same format used at AAA studios and film productions. Each brief pulls from large, hand-curated word banks across jobs, races, genres, themes, cultures, shape language, materials, props, building types, and more.

Every field is lockable. Lock what you like, reroll the rest. Star briefs worth keeping. Combine character and environment into a single scene brief. Rate and annotate your history.

**Fully offline.** No login. No API key. No internet required after the page loads.

---

## Features

- 240+ jobs across everyday, historical, modern, and speculative roles
- 50 fantasy races with visual descriptors
- 170+ themes including animals, bugs, vehicles, and everyday objects
- 160+ building types including grounded and shape-inspired structures
- Field locking — pin any field, reroll everything else
- Single field reroll — hover any cell and hit ↻
- Combine both — generate character and environment as a scene brief
- Starred briefs — persist across sessions via localStorage
- History tab with ratings, notes, and filters
- Custom word banks — add your own entries, pin your favorites
- Ambient music — generative drone, toggleable
- Keyboard shortcuts — R to roll, U to undo, C to combine
- Share URL — encode any brief as a link
- Runs as a PWA — installable to phone home screen

---

## How to use it

Open [conceptvestige.netlify.app](https://conceptvestige.netlify.app) in any browser. That's it.

To run locally: download `index.html` and open it in a browser. No build step, no dependencies, no server.

---

## Contributing

Word bank contributions are welcome. If you have entries that are specific, visual, and genuinely useful to a concept artist — open a pull request adding them to the relevant array in `index.html`.

Good entries:
- specific enough to suggest something drawable
- not redundant with existing entries
- if it's a culture, include a visual shorthand descriptor after the dash

See [CONTRIBUTING.md](CONTRIBUTING.md) for details.
